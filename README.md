# On-mobile-shop

THE  AIM OF  OUR  PROJECT IS TO CREATE A WEB PAGE FOR MOBILE PHONES DISPLAY AND THEIR SPECIFICATIONS  AND TO BUY THEM.THE WEB PAGE IS CREATED USING HTML, CSS AND JAVASCRIPT. 

THE GOAL WAS TO CREATE A WEBSITE TO IMPROVE CUSTOMER KNOWLEDGE AND INFORMATION ON MOBILE PHONES.THIS WAS DONE IN ORDER TO GET BETTER UNDERSTANDING OF THE SKILLS BY COMPARING OF DIFFERENT MOBILE CONCERNS THAT ARE NEEDED IN THE AREA OF WEBSITE DESIGN AND DEVELOPMENT.THERE WERE MANY THINGS THAT WERE TAKEN IN ORDER TO SUBMIT A SUCCESSFUL PROJECT SUCH AS MARKET RESEARCH, IDENTIFYING CUSTOMERS etc..,
